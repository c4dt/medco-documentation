MedCo Connector
===============

2 binaries: server and client

source code:

- `client`: client library (swagger generated)
- `cmd`: binaries (server swagger generated)
- `deployment`: docker-based deployment
- `i2b2`: communication with i2b2
- `medco`: medco query logic implementation
- `models`: swagger models
- `restapi`: server library (swagger generated) + configuration
- `unlynx`: communication with unlynx
- `util`: security and configuration
