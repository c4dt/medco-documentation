Contribution Guidelines¶

Repositories Organisation¶

Code repositories:
  • dev branch: active development
  • master branch: releases
Tags:
  • v*x*.*y*.*zL*

x.y.z: MedCo version L: letter(s) to indicate revision of a version all versions are on docker cloud as well (built as images)

Code organization:
  • deployment folder: contains the deployment related stuff

binaries always hosted somewhere: on docker cloud or on npmjs
